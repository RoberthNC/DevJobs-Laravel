# Pasos de inicialización

1. Clonar el repositorio
2. Instalar los paquetes de node:

-   Con NPM:

```
npm install
```

3. Clonar el archivo .env.example y renombrarlo por .env
4. Modificar las variables de entorno para la configuración de la BD
5. Ejecutar el comando:

```
php artisan serve
```
